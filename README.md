# git-commit-message-convention

| Type          | Description |
|:-------------:|-------------|
| `feat`        | 기능 개발 시 |
| `bug`         | 버그 발생 시 |
| `chore`       | 전반적인 환경 설정 시 |
| `docs`        | 문서 작업 시 |
| `fix`         | 기존 작성된 코드 수정 시 |
| `refactor`    | 리펙토링 시 |
| `test`        | 테스트 관련 |
